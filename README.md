# AI 智能体开发学习项目

这是一个基于 Python 的 AI 智能体开发学习项目，旨在帮助开发者学习和实践 AI 智能体的开发技术。

## 项目结构

```
ai-agent-learning/
├── src/             # 源代码目录
├── tests/           # 测试代码目录
├── docs/            # 文档目录
├── examples/        # 示例代码目录
├── practice01/      # 练习1：LLM 基础调用
├── practice02/      # 练习2：终端聊天界面
├── practice03/      # 练习3：上下文压缩与总结
├── requirements.txt # 依赖项配置
├── setup.py         # 项目安装配置
├── .gitignore       # Git 忽略文件
└── README.md        # 项目说明
```

## 环境配置

1. 确保已安装 Python 3.8 或更高版本
2. 创建并激活虚拟环境：
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # macOS/Linux
   source venv/bin/activate
   ```
3. 安装依赖：
   ```bash
   pip install -e .
   ```

## 快速开始

查看 `examples` 目录中的示例代码，了解如何使用本项目的功能。

## 代码文件说明

### 核心模块

#### `src/__init__.py`
- **功能用途**：定义项目版本号和包初始化
- **教学目标**：了解 Python 包的基本结构和初始化方式

#### `src/cli.py`
- **功能用途**：提供命令行接口，包括打招呼和显示版本功能
- **教学目标**：学习使用 Click 库创建命令行工具，了解命令行参数解析

#### `src/agent/base.py`
- **功能用途**：定义智能体基类，提供思考和执行动作的抽象方法
- **教学目标**：学习面向对象编程中的抽象基类设计，理解接口定义的重要性

#### `src/agent/example.py`
- **功能用途**：实现一个示例智能体，展示如何继承和实现基类方法
- **教学目标**：学习如何实现抽象基类，理解多态的概念

#### `src/utils/helpers.py`
- **功能用途**：提供通用工具函数，如消息格式化和环境变量加载
- **教学目标**：学习如何创建和使用工具模块，提高代码复用性

### 示例代码

#### `examples/basic_agent.py`
- **功能用途**：展示如何创建和使用示例智能体
- **教学目标**：学习智能体的基本使用方法，理解智能体的工作流程

### 练习代码

#### `practice01/llm_client.py`
- **功能用途**：使用标准 http 库访问 OpenAI 兼容协议的 LLM
- **教学目标**：
  - 学习如何读取和解析环境变量
  - 学习使用标准 http 库发送 HTTP 请求
  - 学习如何处理 JSON 数据和 API 响应
  - 理解 OpenAI 兼容协议的基本结构

#### `practice02/chat_interface.py`
- **功能用途**：实现终端聊天界面，支持终端界面输入聊天内容、流式输出、历史聊天记录自动添加到上下文，直到用户 ctrl+c 退出终端，否则一直循环
- **核心功能**：
  - 支持用户在终端中输入聊天内容
  - 实现 LLM 响应的流式输出，提升用户体验
  - 自动将历史聊天记录添加到上下文，支持多轮对话
  - 循环聊天直到用户输入 'exit' 或按 Ctrl+C 退出
- **教学目标**：
  - 学习如何实现流式输出，提升用户体验
  - 学习如何管理聊天历史记录并添加到上下文
  - 学习如何实现终端交互循环
  - 学习如何处理键盘中断（Ctrl+C）
  - 理解 OpenAI 兼容协议的流式响应机制

#### `practice02/tool_chat_client.py`
- **功能用途**：实现带 curl 网络访问功能的终端聊天界面
- **核心功能**：
  - 支持用户输入聊天内容并提交
  - 流式输出 LLM 响应
  - 实现 curl 网络访问功能，可访问网页并返回内容
  - 循环聊天直到用户输入 'exit' 或按 Ctrl+C 退出
- **教学目标**：
  - 学习如何实现工具调用功能
  - 学习如何使用 subprocess 模块执行外部命令
  - 学习如何处理网络访问和超时
  - 理解 LLM 与外部工具的交互机制
  - 学习如何构建具备网络访问能力的智能体

#### `practice02.1/chat_with_tools.py`
- **功能用途**：实现带工具调用功能的终端聊天界面
- **核心功能**：
  - 支持用户输入聊天内容并提交
  - 流式输出 LLM 响应
  - 实现 5 个文件操作工具调用：
    1. list_files：列出目录下的所有文件及其属性
    2. rename_file：修改目录下文件的名字
    3. delete_file：删除目录下的文件
    4. create_file：在目录下新建文件并写入内容
    5. read_file：读取目录下文件的内容
  - 循环聊天直到用户输入 'exit' 或按 Ctrl+C 退出
- **教学目标**：
  - 学习如何实现工具调用功能
  - 学习如何解析和执行工具调用指令
  - 学习如何处理工具执行结果并继续对话
  - 理解 LLM 与外部工具的交互机制
  - 学习如何构建具备工具使用能力的智能体

#### `practice03/chat_with_summary.py`
- **功能用途**：实现带上下文压缩功能的终端聊天界面
- **核心功能**：
  - 支持用户输入聊天内容并提交
  - 流式输出 LLM 响应
  - 自动检测聊天历史记录
  - 当聊天超过5轮或上下文长度超过3k字符时，自动触发聊天记录总结
  - 对前75%左右的对话进行压缩总结，最后30%保留原文
  - 循环聊天直到用户输入 'exit' 或按 Ctrl+C 退出
- **教学目标**：
  - 学习如何检测聊天历史的状态（轮次和长度）
  - 学习如何实现聊天记录的自动总结和压缩
  - 学习如何在保持上下文的同时控制 token 使用量
  - 学习如何构建智能的上下文管理策略
  - 理解长对话场景下的上下文管理优化

#### `practice03/chat_with_5w_and_search.py`
- **功能用途**：实现带 5W 信息提取和聊天历史搜索功能的终端聊天界面
- **核心功能**：
  - 支持用户输入聊天内容并提交
  - 流式输出 LLM 响应
  - 自动检测聊天历史记录，当超过5轮或上下文长度超过3k字符时，自动触发聊天记录总结
  - 从聊天记录中提取 5W 关键信息（谁Who、做了什么事What、什么时候When、在何处Where、为什么要做Why）
  - 将提取的关键信息记录到本地日志文件 D:\chat-log\log.txt（增量更新）
  - 支持通过 "/search" 命令或相关关键词搜索聊天历史
  - 循环聊天直到用户输入 'exit' 或按 Ctrl+C 退出
- **教学目标**：
  - 学习如何从聊天记录中提取关键信息
  - 学习如何实现 5W 规则的信息提取
  - 学习如何将信息持久化到本地文件
  - 学习如何实现聊天历史搜索功能
  - 理解如何构建具备信息提取和搜索能力的智能体
