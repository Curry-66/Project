# 技能管理功能使用说明

## 功能介绍

本项目实现了技能管理功能，能够：
1. 自动读取 `.agents/skills` 目录下的所有技能
2. 提取技能的名称和描述信息
3. 当 LLM 判断需要使用某个技能时，加载技能内容并执行

## 已实现的技能

### notice 技能
- **功能**：用于撰写、修改或润色通知
- **使用场景**：当用户要求撰写通知、修改通知、润色通知时使用
- **规则**：
  - 通知不能以"通知"二字开头
  - 必须冠以"XX部"的前缀，例如"采购部通知"、"宣传部通知"等
  - 如果用户没有告知所在部门，使用"XX部"代替

## 环境配置

1. 确保已创建 `.env` 文件，包含以下环境变量：
   ```
   BASE_URL=http://localhost:1234/v1
   MODEL=lmstudio-community/gemma-2-2b-it-GGUF
   API_KEY=sk-xxx
   ```

2. 确保 `.agents/skills` 目录存在，并且包含技能子目录

## 使用方法

1. 运行聊天助手：
   ```
   python chat_with_skills.py
   ```

2. 与助手对话，当需要撰写通知时，助手会自动调用 notice 技能

## 技能开发指南

### 创建新技能

1. 在 `.agents/skills` 目录下创建新的技能子目录，例如 `new_skill`
2. 在子目录中创建 `SKILL.md` 文件，包含以下内容：
   ```markdown
   ---
   name: new_skill
   description: 技能描述
   ---

   # 技能规则
   技能详细说明...
   ```

3. 重启聊天助手，新技能会自动被加载

### 技能调用格式

当 LLM 需要使用技能时，会输出以下格式：
```
[技能调用开始]
技能名称
[技能调用结束]
```

助手会自动加载技能内容并继续对话。
