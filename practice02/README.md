# AI 聊天助手

一个基于终端的AI聊天助手，支持流式输出、历史聊天记录管理和智能响应过滤。

## 功能特性

- **终端界面**：在命令行中与AI进行实时对话
- **流式输出**：实时显示AI的回答，提供更流畅的用户体验
- **历史记录**：自动保存最近10轮对话，提供上下文理解
- **智能过滤**：移除思考过程和系统指令，只显示核心回答内容
- **错误处理**：提供友好的错误提示和异常处理

## 环境要求

- Python 3.7+
- 一个可用的LLM API服务（如OpenAI、通义千问等）

## 配置方法

1. 在 `practice02` 目录下创建 `.env` 文件
2. 填写以下配置信息：

```env
# LLM API 配置
BASE_URL=http://localhost:8000/v1  # API服务地址
MODEL=your_model_name              # 模型名称
API_KEY=your_api_key               # API密钥
```

## 使用方法

1. 进入 `practice02` 目录
2. 运行聊天助手：

```bash
python chat_interface.py
```

3. 在终端中输入你的问题，助手会实时回答
4. 输入 `exit` 或按 `Ctrl+C` 退出聊天

## 技术实现

- 使用标准 `http.client` 库调用LLM API
- 实现流式输出，提供实时反馈
- 自动管理对话历史，确保上下文理解
- 智能过滤响应内容，移除不必要的信息
- 健壮的错误处理，确保程序稳定运行

## 注意事项

- 确保 `.env` 文件中的API配置正确无误
- 网络连接稳定，以保证API调用的可靠性
- 合理使用历史记录，避免上下文过长导致的性能问题
